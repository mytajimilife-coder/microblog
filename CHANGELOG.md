# Microblog - 更新履歴

## Beta バージョン 1.8.0 (2026-01-04) - 管理画面からのテーマ管理
### 🎉 新機能
- **管理画面テーマ設定**: 管理者パネルからワンクリックでテーマ切り替え。
- **動的テーマエンジン**: 設定をDBに保存し、リアルタイムに適用。

## Beta バージョン 1.7.0 (2026-01-04) - セキュリティと執筆体験の向上
### 🎉 新機能
- **Markdown サポート**: 軽量パーサー内蔵。
- **IP アクセス制御**: 不審なIPのブロック管理機能。
- **自動目次生成**: 見出しから目次を自動作成。

## Beta バージョン 1.6.0 (2026-01-04) - 写稿 API と認証の強化
### 🎉 新機能
- **投稿作成 API**: 外部からの記事投稿をサポート。
- **認証強化**: APIキーとユーザーの紐付け。

## Beta バージョン 1.5.0 (2026-01-04) - REST API システム
### 🎉 新機能
- **REST API エントリポイント**: JSON形式のデータ提供。
- **API キー認証**: 安全な外部アクセスの保証。

## Beta バージョン 1.4.0 (2026-01-04) - プロフェッショナル運用機能
### 🎉 新機能
- **パンくずリスト**: SEO最適化されたナビゲーション。
- **投稿メタデータ**: カスタムフィールドの実装。
- **権限管理 (RBAC)**: ロールベースのアクセス制御。

## Beta バージョン 1.0.0 (初期リリース)
- 基本的なブログ機能の実装
- 日本語環境への最適化
