# Microblog - 日本語専用・高機能ブログシステム

Microblogは、個人の執筆体験を極限まで高めるために設計された、軽量かつパワフルな日本語専用ブログエンジンです。MySQLとPostgreSQLの両方に対応し、最新のセキュリティーとプロフェッショナルな管理機能を備えています。

## ✨ 主な特徴

- **REST API システム**: 外部アプリやモバイルからの投稿・取得が自由自在。
- **マルチテーマ & ダークモード**: 「Sleek Premium」と「Cute Pastel」の2種類のテーマを標準搭載。
- **プロフェッショナル管理画面**: 初心者でもクリック一つでSEO、画像最適化、セキュリティ、テーマを管理可能。
- **高度なセキュリティー**: IPブロック（Firewall）、Brute-force対策、管理者活動ログを完備。
- **執筆支援機能**: Markdownサポート、自動保存（AutoSave）、投稿予約、自動目次(TOC)生成。
- **SEO・SNS最適化**: OGPタグ、JSON-LD構造化データ、日本語スラッグ生成機能を内蔵.
- **パフォーマンス**: ファイルベースの高速キャッシュシステムと画像自動リサイズ。

## 🚀 クイックスタート

1. **アップロード**: Webサーバーにファイルをアップロードします。
2. **インストール**: ブラウザで `install/` ディレクトリにアクセスします。
3. **設定**: 画面の指示に従ってDB情報を入力（自動でテーブルが作成されます）。
4. **執筆開始**: 管理画面からすぐに記事を書き始められます。

## 📂 プロジェクト構造

```
microblog/
├── admin/          # プロフェッショナル管理パネル
├── includes/       # コアロジック (API, SEO, キャッシュ, セキュリティ等)
├── themes/         # テーマフォルダ (Sleek, Cute)
├── config/         # 基本設定
├── install/        # 自動インストーラー
├── api.php         # REST API エントリポイント
└── index.php       # メインフロントエンド
```

## 🛠️ 動作環境

- PHP 7.4 ～ 8.3
- MySQL 5.7+ または PostgreSQL 12+
- GDライブラリ (画像処理用)
